Una variable se puede definir de la siguiente forma:

var 'nombre que quieras' = 'valor';

Ejemplo:

var nombre = 'Edmundo';

var set = true;

Arrays y Objetos

¿Que es un Array?

Un Array es una lista de valores, que puede contener cualquier tipo ya mencionados antes (String,Number,Boolean).

Un Array se puede definir de la siguiente forma:

En este caso lo hemos definido en una variable para poder utilizarla en cualquier lugar

var 'nombre que quieras' = ['valor1', 'valor2', 'valor3', 'valorx'];

Ejemplos:

var alumnos = ['Edmundo', 'Richard', 'Paolo', 'Bruno', 'Ludvin', 'GC'];
var numerosPrimos = [1, 3, 5, 7, 11, 13, 17, 19, 23, ... ];

Y nosotros podemos insertar infinitos valores, y ahora vamos a insertar nuestros propios valores

¿Que es un Objeto?

Un Objeto contiene propiedades y esta conformado por nombre : valor (Sé que esto es dificil de explicar, tambien de entender pero con la practica se logra todo)

Un Objeto se puede definir de la siguiente forma:

En este caso tambien lo hemos definido en una variable para poder utilizarla en cualquier lugar

var 'nombre que quieras' = { name: 'valor', name2: 'valor2', name3: 'valor3', namex: 'valorx' }

Donde valor puede contener cualquier valor sea tipo string/number/boolean

Ejemplos:

var Edmundo = { name: 'Edmundo', lastname: 'Acosta', age: 18, carrera: 'DDMD' }
var Auto = { marca: 'Audi', valor: 500, velocidad: 80, peso: 90, Dueño: 'Bruno' }

Este codigo lo entenderan mejor si lo ven en main.js pues ahi sale como se estructura principalmente un objeto
